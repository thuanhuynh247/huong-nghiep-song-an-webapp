## Before you publish

1. Set the real public URL by running:

```powershell
python scripts/set_public_url.py https://your-real-public-domain/
```

2. Review the launch checklist in `docs/official-launch-checklist.md`.
3. Push to the production repository and let the Pages workflow deploy the app.
